## 📌 重要提醒

### ⚠️ 使用前必读
- [ ] 已设置 OpenAI API Key
- [ ] 已配置 rpvm_config.yaml 中的路径
- [ ] 已准备 E5 模型和索引文件
- [ ] 已准备数据集
- [ ] 运行过 test_rpvm.py 验证环境

### 💰 成本注意
- OpenAI API 按 token 计费
- 建议先用小样本测试（--num_samples 5）
- 监控 API 使用量
- 预估成本：每样本约 5-20 次 API 调用

### 📁 文件管理
- 定期备份实验结果
- 每次实验保存配置文件副本
- 注意输出目录的磁盘空间
- 中间数据文件可能较大
