RPVM 配置说明
================

✅ 已完成的配置：
1. E5模型路径: examples/quick_start/e5-base-v2
2. OpenAI API配置: 从项目根目录的.env文件自动读取
   - OPENAI_API_KEY
   - OPENAI_BASE_URL

⚠️ 还需要准备的：
1. 数据集文件
   - 位置: dataset/hotpotqa/ 或 dataset/2wikimultihopqa/
   - 参考FlashRAG文档准备数据集

2. 检索索引文件（需要构建）
   - index_path: indexes/e5_Flat.index
   - corpus_path: indexes/general_knowledge.jsonl
   - 使用FlashRAG提供的脚本构建索引

快速开始：
1. 确保已安装 python-dotenv: pip install python-dotenv
2. 运行测试: python test_rpvm.py
3. 简单示例: python simple_example.py
4. 正式实验: python run_rpvm_exp.py --dataset_name hotpotqa --num_samples 5

当前配置文件：
- rpvm_config.yaml: 主配置文件（已更新E5路径）
- .env: OpenAI API配置（项目根目录）

注意事项：
- .env文件会被自动加载，无需手动设置环境变量
- 如果python-dotenv未安装，仍可通过系统环境变量设置
- 所有路径都相对于项目根目录（flashRAG/）
